# Lays Express API Scaffold

This project is a scaffold for a Node.js REST API using Express and MongoDB (Mongoose), with ES modules and dotenv for environment configuration.

## Features
- Express server setup
- MongoDB connection via Mongoose
- ES module syntax
- Environment variable support
